# Siriel Adventures

Arcade game.

More information https://georgik.rocks/siriel-game/

## Siriel 3.5 Dosbox edition

Requirements: dosbox

Controls: Keyboard (arrows, spaces, enter), Gamepad

Start the game:

```
./run-siriel-3.5-en_US.sh
```

### Gamepad support

Change value in the file `BIN/SIRIEL3.CFG`:

```
[JOYSTICK]=YES
```
